# Workin

<h3>1. 기획 배경</h3>
- 2차 프로젝트로 진행한 워크인을 스프링을 이용한 웹사이트로 구현하고자 함<br>
- 하나의 회사가 아닌 여러 회사가 사용할 수있도록 사용 범위를 늘리고자 함<br>

<h3>2. 기획 목표</h3> 
- 직원과 관리자 모드로 나뉘어 관리<br>
- 직원모드 : 자신의 일정을 확인, 일정 신청, 출퇴근<br>
- 관리자모드 : 직원들의 일정을 확인하고 요청을 수락, 거절<br>
